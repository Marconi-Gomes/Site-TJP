Java

TJPE

Em segundo lugar, após instalar o Java, clique no botão de Download, aguarde o download ser concluído e execute o instalador como Administrador até finalizar a instalação. Depois de instalar, insira o Token no PC/Notebook e siga para o próximo programa abaixo..

Token

Por fim, após conectar o token, clique no botão de Download, aguarde o download ser concluído e execute o instalador como Administrador. Siga o passo a passo da instalação e, ao concluir, reinicie o PC para atualizar o driver. Após reiniciar, configure o token. Caso tenha alguma dúvida, assista ao vídeo abaixo.
